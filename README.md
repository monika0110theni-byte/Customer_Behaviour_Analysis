🛍️ Customer Shopping Behavior Analysis
📌 Project Overview
This end-to-end Data Analytics project analyzes customer shopping behavior to uncover purchasing patterns, customer preferences, and sales trends. The project combines Python, PostgreSQL, and Power BI to transform raw data into actionable business insights through data analysis, visualization, and reporting.

🎯 Project Objectives
Understand customer purchasing behavior.
Identify top-performing product categories.
Analyze customer demographics and spending patterns.
Discover trends that influence sales performance.
Build an interactive dashboard for business decision-making.
📊 Dataset Information
The dataset contains customer shopping and transaction details, including:

Customer ID
Age
Gender
Category
Item Purchased
Purchase Amount (USD)
Subscription Status
Previous Purchases
Payment Method
Location
